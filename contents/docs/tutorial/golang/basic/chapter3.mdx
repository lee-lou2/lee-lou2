# Golang: 구조체와 인터페이스 튜토리얼

## 구조체 (Structs)

구조체는 여러 개의 필드를 묶어 하나의 집합체로 표현하는 방법입니다.

### 구조체 정의하기

```go
type Person struct {
        Name string
        Age  int
}
```

### 구조체 인스턴스 생성하기

```go
john := Person{Name: "John", Age: 30}
```

### 구조체 필드 접근하기

```go
fmt.Println(john.Name) // 출력: John
fmt.Println(john.Age)  // 출력: 30
```

## 인터페이스 (Interfaces)

인터페이스는 메서드 시그니처의 집합으로, 객체가 어떤 메서드를 구현해야 하는지 나타내는 방법입니다.

### 인터페이스 정의하기

```go
type Speaker interface {
        Speak() string
}
```

### 인터페이스 구현하기

구현하려는 인터페이스 메서드를 모두 포함하여 구조체를 정의해 인터페이스를 구현합니다.

```go
func (p Person) Speak() string {
        return fmt.Sprintf("Hi, my name is %s, and I'm %d years old.", p.Name, p.Age)
}

// Person 구조체는 Speaker 인터페이스를 구현했습니다.
```

### 인터페이스를 통한 메서드 호출

```go
func CallSpeaker(speaker Speaker) {
        fmt.Println(speaker.Speak())
}

CallSpeaker(john) // 출력: Hi, my name is John, and I'm 30 years old.
```

이 튜토리얼은 Golang에서 구조체와 인터페이스를 설명하고 사용하는 방법을 다룹니다. 이를 통해 사용자 지정 타입을 만들거나 객체지향 프로그래밍 패턴을 적용할 수 있습니다.
