# 고 기초

고(Go)는 구글에서 개발한 프로그래밍 언어입니다. 이번 장에서는 고 언어의 기초적인 내용을 다룹니다.

## 변수

고 언어에서 변수를 선언할 때에는 `var` 키워드를 사용합니다. 변수의 타입은 자동으로 추론되며, 타입을 명시적으로 지정할 수도 있습니다.

```go
var a = 10
var b string = "hello"
```

## 함수

고 언어에서 함수를 정의할 때에는 `func` 키워드를 사용합니다. 함수의 반환값은 함수의 매개변수 뒤에 타입을 지정하여 정의할 수 있습니다.

```go
func add(a int, b int) int {
    return a + b
}
```

## 제어문

고 언어에서 제어문은 C 언어와 유사합니다. `if`, `for`, `switch` 등의 키워드를 사용하여 제어문을 작성할 수 있습니다.

```go
if a > 10 {
    fmt.Println("a is greater than 10")
} else {
    fmt.Println("a is less than or equal to 10")
}

for i := 0; i < 10; i++ {
    fmt.Println(i)
}

switch a {
case 1:
    fmt.Println("a is 1")
case 2:
    fmt.Println("a is 2")
default:
    fmt.Println("a is neither 1 nor 2")
}
```

## 포인터

고 언어에서 포인터를 사용할 때에는 `*` 연산자를 사용합니다. 포인터를 선언할 때에는 `*` 연산자를 변수 이름 앞에 붙여서 사용합니다.

```go
var a int = 10
var b *int = &a

fmt.Println(*b) // 10
```

## 구조체

고 언어에서 구조체를 정의할 때에는 `type` 키워드를 사용합니다. 구조체의 필드는 `name type` 형식으로 정의할 수 있습니다.

```go
type Person struct {
    Name string
    Age int
}

var p Person = Person{"John", 30}
fmt.Println(p.Name) // John
```
