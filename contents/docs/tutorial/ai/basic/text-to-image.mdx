# 📄 Text-To-Image

## ✏️ 프롬프트 작성 방법

### 1. Prompt / Negative prompt

:::tip
긍정 프롬프트와 부정 프롬프트를 잘 활용하자!
:::

대부분의 Text To Image 모델에는 긍정 프롬프트와 부정 프롬프트가 존재합니다.
긍/부정 프롬프트를 잘 활용하면 좀 더 원하는 이미지를 생성할 수 있습니다.
최초 프롬프트로 생성한 후 원하는 방향을 긍/부정 프롬프트에 하나씩 넣어가며 생성해보세요.

### 2. 가중치

:::tip
가중치를 이용해 원하는 방향을 지정할 수 있습니다
:::

**가중치**란 프롬프트의 영향력을 증가시키는 것입니다.
(가중치의 기본 값은 1입니다. 모든 프롬프트는 1의 가중치를 갖습니다.)

가중치를 다루는 방법은 더 많이 있긴 한데 가장 중요한 한 가지 방법에 대해서만 말씀드리겠습니다.

example_prompt에 가중치를 부여하는 방법은 다음과 같습니다.

(example_prompt:1.5)

하지만 하나의 프롬프트에 높은 영향력을 부여하고 싶다고 가중치를 너무 높게 설정하시면 안 됩니다.
그림이 망가져 버립니다.

보통 0.8~1.5 사이로 가중치를 설정하시는 것을 추천드립니다.


### 3. 문장형/태그형 프롬프트

:::tip
태그형과 문장형을 적절히 사용하여 프롬프트를 작성하자!
:::

외모, 상태, 배경 등은 **태그형 프롬프트**<br />
구도, 상황, 행위 묘사 등은 **문장형 프롬프트**

태그형 프롬프트 예제 : (black_hair),(white_background) 등등<br />
문장형 프롬프트 예제 : (standing on the table),(looking at window) 등등
