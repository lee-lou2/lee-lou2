## 🐥 예시

```python
gpg --keyserver keyserver.ubuntu.com --recv [KEY]

gpg --export --armor [KEY] | sudo apt-key add -

wget -q -O - https://packages.cloudfoundry.org/debian/cli.cloudfoundry.org.key | sudo apt-key add -
```
