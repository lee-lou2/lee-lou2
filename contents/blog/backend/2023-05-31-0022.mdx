## ⭐ 키워드

| 키워드       | 설명                     |
|-----------|------------------------|
| goroutine | 고에서 제공하는 경량 쓰레드        |
| thread    | 동시성 / 병렬성을 제공하는 시스템 방식 |

## 🐥 차이

|                | Goroutine                   | Thread                                           |
|----------------|-----------------------------|--------------------------------------------------|
| 메모리 비용         | 2KB                         | 1MB                                              |
| 설치, 철거 비용      | 런타임에서 만들어지고 파괴되는 작업들이 매우 저렴 | OS로부터 리소스를 요청하고 작업이 끝나면 리소스를 돌려줘야 함              |
| Content 스위칭 비용 | 오직 3개의 레지스터만이 save/restore  | context switching이 일어날 시 16개의 레지스터가 save/restore |
